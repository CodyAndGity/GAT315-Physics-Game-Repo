# GAT315 Physics Game Repo

A physics-based driving game built for GAT315. Navigate a truck through ramps, obstacles, and other hazards while keeping control of your vehicle and reaching the goal.

## Overview

This project focuses on physics-driven movement, obstacle navigation, and a simple arcade-style challenge loop. The game tracks player deaths on screen, giving you a quick way to see how many times you have crashed or fallen.

## Features

- Physics-based vehicle movement
- Obstacle-course style level design
- On-screen death counter
- Custom visual effects and stylized environment
- Original game assets and imported audio

## Getting Started

### Requirements
- Unity installed on your machine
- A supported version of the Unity Editor for the project

### Run the Project
1. Clone this repository.
2. Open the project folder in Unity Hub.
3. Let Unity finish importing assets and packages.
4. Open the main scene.
5. Press **Play** to start testing the game.

## Controls

Default controls may vary depending on the build, but the game is designed around standard vehicle controls such as:

- **W / Up Arrow** — Accelerate
- **S / Down Arrow** — Reverse / brake
- **A / Left Arrow** — Steer left
- **D / Right Arrow** — Steer right

If your project uses custom input bindings, update this section to match your current setup.

## Gameplay

Your goal is to drive carefully across the course and avoid falling, flipping, or crashing into hazards. The further you progress, the more important momentum and timing become.
